# 나의 자동화 시스템: 쇼핑숏폼 제작 도우미

## 1. 목표

- 무엇을: 쇼핑숏폼 제작 플로우 전반에 걸쳐 필요한 부분 자동화
- 현재: 레퍼런스 분석, 촬영, 편집, 업로드까지 5~6시간 소요
- 목표: 촬영시간을 제외하고 2시간으로 단축

## 2. Sub-agents 구성

- 쇼츠 PD: 숏핑 쇼츠 영상 제작 전체 워크플로우를 오케스트레이션하는 메인 PD 에이전트
- 트렌드 리서처: 구독 채널 분석 및 트렌드 제품 발굴
- 레퍼런스 분석 에이전트: 선정된 제품 관련 레퍼런스 영상 분석
- 대본 에이전트: 미리 정의된 지침에 따라 쇼츠 대본을 작성
- TTS 에이전트: Typecast API 를 사용해서 TTS 음성 생성, SRT 자막 생성 & 싱크
- 소스 수집 에이전트: 대본의 각 컷에 맞는 이미지/영상 소스 수집 가이드 & AI 이미지 프롬포트 작성
- 편집가이드 에이전트: 대본과 소스영상을 분석해서 타임라인 가이드 및 체크리스트 생성
- 업로드 매니저: 완성된 영상의 채널 업로드에 필요한 모든 텍스트 정보 생성
- 썸네일 메이커: 쇼츠 썸네일 이미지 생성

## 3. Skills (CLAUDE.md)

- 품질 기준: 최근 바이럴된 영상에 등장한 제품, 미리 제공한 포맷에 맞는 대본 작성
- 포맷: 3종 묶음 또는 1종 집중 분석 영상, 자세한 인기 숏츠 기준 및 대본 포맷은 미리 제공(예시 포함), 미리 제공한 포맷에 맞는 업로드 정보 생성

## 4. MCP 연결

- 입력: 각 단계의 산출물 생성
- 저장: 노션 '쇼핑숏폼 컨텐츠' DB
- 공유: Slack #shortping 채널

## 5. 트리거 프롬프트

- /research-product -> 트렌드 리서치 & 제품발굴 시작
- /make-video -> 영상 제작 플로우 시작
