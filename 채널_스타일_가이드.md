## 🖼 썸네일 스타일

### 텍스트 구성
- **3줄 구조** (한 줄 6~9자)
- **부분 강조 패턴** — 한 줄 안에서 **브랜드명·핵심 임팩트 단어**만 빨강, 나머지는 흰색
- 마지막 줄은 **CTA형 행동 유도** ("이건 꼭 사야해", "안 사면 후회" 등) — 흰색 단색

#### 강조 패턴 예시 (2026-05-09 기준)
```
[다이소가] 미쳤어요       ← 브랜드명 빨강 + 술어 흰색
[전통] 시리즈            ← 핵심 카테고리 빨강 + 서브 흰색
이건 꼭 사야해           ← CTA 흰색 단색
```

### 텍스트 스타일
| 항목 | 값 |
|------|-----|
| 폰트 | **에스코어드림 9 Black** 또는 **Apple SD Gothic Neo Heavy** |
| 폰트 두께 | 가장 굵은 단계 (Black / Heavy / ExtraBold) |
| **강조 색상** | **빨강 #FF1F1F** (브랜드명·핵심 단어 부분 강조) |
| 본문 색상 | **흰색 #FFFFFF** |
| 테두리 | 검정 (#000000), **6~7px 두께** |
| 그림자 | 검정 50%, 거리 4~5px |
| 정렬 | 가운데 정렬 |
| 줄 간격 | line-height 1.1 (좁게) |
| 위치 | 화면 중앙 (Y축 35~60% 영역) |
| 회전 | 살짝 (-2도 또는 0도) |

### 배경 이미지
- **제품 사용 시연 컷** 활용 (영상에서 추출한 정지 프레임)
- 어두운 그라데이션 오버레이 추가 (텍스트 가독성)
- 베란다·인테리어·라이프스타일 분위기 이미지 선호

### thumbnail_generate.py 호출 예시
```bash
python3 thumbnail_generate.py "workspace/{폴더}" \
  --line1 "{핵심 후킹}" \
  --line2 "{보완 1}" \
  --line3 "{CTA}" \
  --accent "#FF1F1F"
```

## 📺 영상 자막 스타일

### 핵심 원칙
- ✅ **단색 흰색** (강조 색상 사용 안 함)
- ✅ 두꺼운 검정 테두리로 가독성 확보
- ✅ 굵은 고딕 폰트 (블랙/엑스트라볼드)
- ❌ 노란색·빨간색 등 강조 색상 사용 금지 (영상 자막은 단순하게)

### 텍스트 스타일
| 항목 | 값 |
|------|-----|
| 폰트 | **에스코어드림 9 Black** 또는 **Apple SD Gothic Neo Heavy** |
| 폰트 두께 | Heavy / Black (가장 굵은 단계) |
| 색상 | **흰색 #FFFFFF 단색** |
| 테두리 | 검정 (#000000), **3~4px 두께** |
| 그림자 | 검정, 살짝 (1~2px) |
| 정렬 | 가운데 정렬 |
| 위치 | **화면 위쪽 ~30~38%** (Y축 약 600~720px) |
| 크기 | 한 줄 4~9자가 화면 너비의 60~70% 차지 |

### ffmpeg burn-in 명령 (참고값)
```bash
ffmpeg -i input.mp4 \
  -vf "subtitles=subtitle.srt:force_style='FontName=Apple SD Gothic Neo,FontSize=18,Bold=1,PrimaryColour=&H00FFFFFF,OutlineColour=&H00000000,BackColour=&H00000000,Outline=3,Shadow=1,MarginV=200,Alignment=2'" \
  output.mp4
```

### 자막 위치 검증
사용자 영상 분석:
- 0초 (썸네일): 화면 중앙
- 2초 (안전난간이라고요?): 화면 위쪽 약 30%
- 4초 (안 쓸 땐 튼튼한): 화면 위쪽 약 28%
- 6초 (필요할 땐 쏙): 화면 위쪽 약 25%
- 10초 (햇빛이랑 바람): 화면 위쪽 약 30%
- 14초 (다 쓰면 다시 착): 화면 위쪽 약 30%
- 18초 (제품정보는): 화면 위쪽 약 25%

→ 자막 위치는 **화면 위쪽 25~32%** 범위에 일관되게 배치

## ✏️ 폰트 다운로드/설정

### 추천 폰트 우선순위
1. **에스코어드림 9 Black** ⭐ 가장 권장
   - 다운로드: https://noonnu.cc/font_page/126
   - 굵고 깔끔한 느낌, 한국어 숏폼에 최적
2. **Apple SD Gothic Neo Heavy** (macOS 기본)
   - 시스템 폰트, 별도 설치 불필요
   - 경로: `/System/Library/Fonts/AppleSDGothicNeo.ttc`
3. **Pretendard Black**
   - 다운로드: https://github.com/orioncactus/pretendard
4. **나눔스퀘어라운드 ExtraBold**
   - 살짝 라운드 느낌

### CapCut 적용
1. CapCut → 텍스트 → 폰트 선택
2. 시스템 폰트에서 위 폰트 선택
3. 두께: Bold/Black 우선

### ffmpeg subtitles 적용
`force_style='FontName=Apple SD Gothic Neo'` 또는 사용자 설치 폰트명 명시.
