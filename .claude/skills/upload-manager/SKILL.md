# 업로드 매니저

## 역할

완성된 영상의 YouTube 업로드에 필요한 모든 텍스트 정보를 준비한다.
SEO 최적화된 제목, 설명란, 해시태그, 썸네일 문구를 작성하고
쿠팡 파트너스 링크 등록 안내를 제공한다.

---

## 입력

- `workspace/{폴더}/03_script.md` — 대본 (제품명, 특징)
- `workspace/{폴더}/01_products.md` — 제품 정보 (가격, 구매처)

---

## 실행 단계

### Step 1: 제목 후보 3개 작성

YouTube 쇼츠 제목 최적화 원칙:
- 20~30자 이내
- 핵심 키워드 앞부분 배치
- 숫자/감탄사 활용
- 클릭 유도 문구

후보 예시 패턴:
- "다이소 {가격} {제품} 샀더니 이렇게 됨 🤯"
- "삶의 질 올라가는 {카테고리} 꿀템 {N}가지"
- "{제품}이 {가격}이라고? 다이소 신상 꿀템"

### Step 2: 설명란 + 해시태그 작성 (복사용 한 덩어리)

설명란과 해시태그를 하나의 코드 블록으로 합쳐서 바로 복사할 수 있도록 작성한다.

```
{후킹 문장 1줄}
{제품 핵심 소개 1줄}

✅ {특징1}
✅ {특징2}
✅ {특징3}
✅ {특징4}

📌 제품 정보는 프로필 링크에서 {N}번 검색하세요

#쿠팡꿀템 #{키워드1} #{키워드2} #{키워드3} #{키워드4}
```

### Step 3: 해시태그 규칙

- **5개만** 선정 (많이 넣지 않기)
- **필수**: #쿠팡꿀템
- 나머지 4개: 제품 카테고리 + 핵심 키워드

### Step 4: 썸네일 문구 작성

3줄 구성, 짧고 강렬하게:
```
{첫째줄: 후킹 키워드}
{둘째줄: 제품 핵심 특징}
{셋째줄: 가격 또는 수량}
```

예시:
```
냄비 필요없는
만능 접시
주방 끝판왕
```

```
## 🛒 쿠팡 파트너스 링크 등록 방법

1. https://partners.coupang.com 접속
2. 제품 검색: "{제품명}"
3. 링크 생성 → 복사
4. 위 설명란의 `{쿠팡 파트너스 링크 자리}`에 붙여넣기

- [ ] 영상 파일 준비 (MP4, 1080x1920)
- [ ] 썸네일 이미지 준비 (JPG, 1080x1920)

- [ ] 공개 설정: 즉시공개 또는 예약
- [ ] 배경음악 추가 (YouTube 오디오 라이브러리)
```
