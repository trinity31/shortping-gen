# 업로드 매니저

## 역할

완성된 영상의 YouTube 업로드에 필요한 모든 텍스트 정보를 준비한다.
SEO 최적화된 제목, 설명란, 해시태그, 썸네일 문구를 작성하고
쿠팡 파트너스 링크 등록 안내를 제공한다.

---

## 입력

- `workspace/{폴더}/03_script.md` — 대본 (제품명, 특징)
- `workspace/{폴더}/01_products.md` — 제품 정보 (가격, 구매처)

---

## 실행 단계

### Step 1: 제목 후보 3개 작성

YouTube 쇼츠 제목 최적화 원칙:
- 20~30자 이내
- 핵심 키워드 앞부분 배치
- 숫자/감탄사 활용
- 클릭 유도 문구

후보 예시 패턴:
- "다이소 {가격} {제품} 샀더니 이렇게 됨 🤯"
- "삶의 질 올라가는 {카테고리} 꿀템 {N}가지"
- "{제품}이 {가격}이라고? 다이소 신상 꿀템"

### Step 2: 설명란 + 해시태그 작성 (복사용 한 덩어리)

설명란과 해시태그를 하나의 코드 블록으로 합쳐서 바로 복사할 수 있도록 작성한다.

```
{후킹 문장 1줄}

제품정보는 프로필 링크 클릭 → {N}번 검색하세요!

{제품 핵심 소개 1~3줄}

✅ {특징1}
✅ {특징2}
✅ {특징3}
✅ {특징4}

#쿠팡꿀템 #{키워드1} #{키워드2} #{키워드3} #{키워드4}
```

### ⚠️ "검색하세요!" 문장 위치 — 반드시 상단

검색 유도 문장은 **반드시 후킹 문장 바로 아래(설명란 두 번째 줄)**에 배치한다.
- ❌ 설명란 끝(해시태그 직전)에 배치 금지
- ❌ 특징·타겟 리스트 사이에 배치 금지
- ✅ 후킹 문장 → 검색 유도 → 본문(소개·특징·타겟) → 해시태그 순서 고정

**이유:** YouTube Shorts 시청자가 설명란을 펼치는 순간 가장 먼저 행동 유도(CTA)를 보게 하기 위함. 본문을 다 읽지 않고도 즉시 프로필 링크로 이동 가능.

### ⚠️ 설명란 절대 금지 사항

**다음 정보를 설명란에 절대 직접 노출하지 않는다:**
- ❌ 가격 (예: "29,490원", "1만원대")
- ❌ 구매 링크 (예: "https://www.coupang.com/...", "쿠팡 링크: ...")
- ❌ 쿠팡 파트너스 단축 링크 (예: "https://link.coupang.com/...")
- ❌ 별도 "구매처 / 쇼핑몰" 안내 문구

**이유:** 모든 구매 정보는 채널의 **프로필 링크 모음 페이지(예: linktree, litt.ly 등)**에 영상 번호별로 정리되며, YouTube 설명란에는 **"제품정보는 프로필 링크 클릭 → {N}번 검색하세요!"** 한 줄로만 유도한다.

### 영상 번호 {N} 처리

- 영상 번호는 채널의 누적 영상 카운터 (예: 81번, 82번, 83번 ...)
- 사용자에게 **이번 영상의 번호**를 명시적으로 묻는다:
  > "이번 영상은 몇 번인가요? (이전 영상 다음 번호로 추정해도 될까요?)"
- 추정 가능한 경우(이전 폴더의 업로드_정보.md에서 마지막 번호 확인): **이전 번호 + 1**을 제안하고 사용자 확인 받기
- 사용자 확정 전까지 `{N}` placeholder로 둔다

### Step 3: 해시태그 규칙

- ⚠️ **설명란(복사용 블록)에 넣는 해시태그는 최대 5개까지만** — 절대 초과하지 않는다
- **필수**: #쿠팡꿀템
- 나머지 4개: 제품 카테고리 + 핵심 키워드
- 추가 해시태그(10~15개 전체 리스트)를 별도 섹션으로 내보내는 것 금지 — 설명란 5개가 유일한 해시태그 블록

### Step 4: 썸네일 문구 작성

3줄 구성, 짧고 강렬하게:
```
{첫째줄: 후킹 키워드}
{둘째줄: 제품 핵심 특징}
{셋째줄: 가격 또는 수량}
```

예시:
```
냄비 필요없는
만능 접시
주방 끝판왕
```

## 🛒 프로필 링크 페이지 등록 방법 (YouTube 설명란 X)

1. https://partners.coupang.com 접속
2. 제품 검색: "{제품명}"
3. 링크 생성 → 단축 링크 복사
4. **프로필 링크 모음 페이지** (linktree / litt.ly / 노션 등)에 다음 형식으로 추가:
   ```
   {N}번. {제품명 또는 영상 키워드}
   {쿠팡 파트너스 단축 링크}
   ```
5. ⚠️ YouTube 설명란에는 **링크·가격 직접 노출 금지** — "제품정보는 프로필 링크 클릭 → {N}번 검색!"으로만 유도
