## 대본 작성 지침

### 오프닝 후킹
| 규칙 |
| --- |
| **"삶의 질 확 올라가는 [카테고리] 꿀템 [N]가지"** 로 시작 |
| 예시: "삶의 질 확 올라가는 욕실 꿀템 3가지" |
| 예시: "삶의 질 확 올라가는 주방 꿀템 5가지" |
| 예시: "삶의 질 확 올라가는 여행 꿀템 3가지" |

### 대본 규칙
| 규칙                                                 |
| -------------------------------------------------- |
| 궁금하게 만드는 후킹 문장으로 시작                                |
| 한 제품당 최대한 짧게(10~12초 권장)                            |
| 반복되는 말 하지 않기                                       |
| 제품의 핵심 기능만 빠르게 보여주기                                |
| 빌드업은 최소한으로 하고 본론으로 바로 들어가기                         |
| 최대한 말 끝내지 않고 연결어를 사용해서 계속 보게 만들기                   |
| 문제 제시 & 해결책 제시 대신 그냥 바로 보여주고 (~할 필요없이 ~면 되요) 짧게 쓰기 |

### 구조 (살림남 스타일, ~28초)
| 구간  | 시간     | 내용                        |
| --- | ------ | ------------------------- |
| 오프닝 | 0~1초   | 후킹 문구 + 제품 빠른 프리뷰         |
| 제품1 | 1~9초   | "첫 번째," + 핵심 기능 + 감탄사     |
| 제품2 | 9~18초  | "두 번째," + 핵심 기능 + 감탄사     |
| 제품3 | 18~25초 | "세 번째," + 핵심 기능 + CTA     |
| 클로징 | 25~28초 | 마무리 멘트 + "제품정보는 프로필 링크에!" |

### 톤 & 감탄사
| 규칙 |
| --- |
| 친구한테 말하듯 캐주얼한 톤, 단 **모든 문장은 존댓말(~요, ~세요, ~에요) 필수** |
| **반말(~해, ~써봐, ~해봐) 절대 금지** |
| 넘버링: "첫 번째", "두 번째", "세 번째" |
| 감탄사: "끄떡없고요", "최고입니다", "편리하답니다", "겟 하세요!" |
| 연결어미: ~고요, ~되니까, ~인데, ~해서 |
| **클로징 CTA 고정**: "제품정보는 프로필 링크에 있어요" |

### 화면 구성
| 규칙 |
| --- |
| 💚 화면 구성을 컷별 초 단위로 상세 지정 |
| 컷당 1~2초, 빠른 전환 |
| 제품 등장 → 핵심 기능 시연 → 결과/효과 순서 |
